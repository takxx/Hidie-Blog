[ドキドキ鳥取] Day6：水木しげる記念館

終於到達今日最最最主要嘅地方

水木しげる記念館（水木茂紀念館）

呃！今日出沒嘅係鬼太郎！
一直望住其他人嘅妖菓目玉おやじ
真係為食嘅佢呢呢呢～

好nice嘅其實佢（同個角色一樣）

仲會一路搞個頭（髮）

門前有小時候嘅鬼太郎父子銅像

洗手間牌都係角色們樣式

臨近聖誕就係聖誕造型

「奇怪地」用咗まんが王国とっとり満喫周遊パス換入場券
雖然唔係C9數不過都要計一計：
料金700円；JR pass減400円 = 300円（加送紀念品！）；
鳥取藩乗放題手形bus pass減100円 = 600円
所以，呢個周遊pass一定要買！

紀念品係非賣品嘅A5 file

有分可攝影同不可攝影區域
望住爸爸嘅指示就OK

水木しげる 漫画ワールド
個館順路係先講水木先生嘅著作

主要角色嘅秘密（噓～）

仲有水木先生响世界國地冒險時搜羅嘅玩兒

鬼太郎嘅書同official玩具

reading together

真跡壁畫啊！

跟住行，就係ねぼけ人生の間
係詳細講述水木先生嘅生平

水木しげるの仕事部屋
整齊過青山剛昌先生嘅好多哈哈

のんのんばあとオレ
去睇水木先生由小時候開始創作妖怪世界嘅起點

會眨眼嘅牆壁怪

其實就係啲一直以來嘅古老妖怪傳說

鍾意响夜晚行行吓嘅べとべとさん

妖怪洞窟
呢啲就係水木先生筆下嘅妖怪

仲有得窺探妖怪戶嘅生活

3D畫影相位，一齊飛上天上

企画展示『アニメ「ゲゲゲの鬼太郎」展』
2018年起鬼太郎animation嘅50週年
而家日本電視嘅係第6輯
呢度展示返咁多年嘅畫作同故事嘅變遷
有得坐低睇咁多輯嘅片頭&片尾曲
一路「ゲッゲッ ゲゲゲのゲイ」同「カランコロン カランカランコロン」
咩回憶都返晒嚟

bye～
