[東京箱根食買浸] Day6：新幹線 x skyliner x きつねうどん

搭乘巴士到新幹線經上野去成田空港 呵呵呵～ 搭新幹線 等車中，對面那輛也是新幹線 見它也是來得快去得快 我坐這架（興奮興奮） 很快便到達目的地（相比romancecar） 日本的快車都很穩定 因為我要坐最快到達上野的車 但剛剛這輛新幹線還差一個站才到上野 要再轉這輛快快的車 車程只有幾分鐘，話說我還未找到我的位置已到站 XD 到了上野站，就是搭這skyliner去成田 雖然NEX由東京去機場是最快 但從箱根出發這樣搭好像更快而且可試塔不同的車 （貪新鮮的人～） 聽說京城電鐵也有車到機場 但看到這輛車感覺似三天後才會到達 XDDD 搭這搭那的，到達成田空港已經是下午 機場裡好像沒啥好吃的 便試試這個在漫畫裡常看到的「きつねうどん」 果然，沒有特別 然後在機場再作最後衝刺，要上機了（哭）

[東京箱根食買浸] Day5：箱根ホテル小涌園 紅葉 特選会席

最後一晚在日本的晚餐，超～ 豪～ 食～！ 等食中，很清雅 お酒：今晚喝熱的 前菜：精緻，中間那隻杯中的魚很清爽 釜飯蒸煮中 刺身：蝦、赤貝、とろ⋯⋯ 全部都很鮮甜味美（大愛） 御椀：淺湯汁浸的魚鮮甜滑溜 沒有過熟令魚肉鞋口 煮物：魚的樣子比較醜 但魚肉很甜也不乾身，煮汁味道少鹹帶甜 牛フィレステーキ：五成熟的煎牛扒 開蓋時香氣撲鼻，煎封了的牛扒肉汁被鎖住 肥瘦適中肉腍而不膩，沒有不好吃的道理 焼き魚：不是不好吃只是對比燒牛的話便沒有那麼香而已 （其實這個燒鯖魚很不錯吃的說） 茶碗蒸し：這杯蛋 超～～ 滑～～～ 是整個日本行程中吃過最滑的蛋，好好味 釜飯：是菇飯，味道很香很清淡 但沒有食蟹釜飯那麼驚艷 反之隔離位的魚生飯很像更吸引（閃亮閃亮）

[東京箱根食買浸] Day4：焼肉＠新宿

從御殿場回到新宿已是七時多，要先卸下戰利品 然後，當然是去找吃的⋯⋯ 吃焼肉去！ 肉呀肉呀～ 牛扒牛仔肉牛舌 樣樣都新鮮好吃，樣樣都鍾意 :p 備長炭烤爐～ 牛內臟的前菜，爽口不太鹹也沒腥味 味道很好，佐酒更是一流 這餐很好吃，唯獨怎樣也記不起店名 新宿這一帶有很多焼肉店而且每間也看似很美味 其實在新宿車站下車時很想吃車仔拉麵（每晚行經這裡都想試） 可惜當時實在有太多東西想先回酒店 從酒店出來卻去了吃焼肉⋯⋯ 吃完焼肉後回來車仔檔已收工 （別問為甚麼吃完焼肉還可以吃拉麵好嗎） 今天已是最後一晚在東京了，唯有下次再來吃拉麵吧～ -———————————————- 延伸閱讀：東京箱根食買浸 (6D5N)
